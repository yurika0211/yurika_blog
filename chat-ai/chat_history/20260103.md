首先，假设我们有一个简单的结构体定义，比如：
type Person struct {
    Name string
    Age  int
}

1. 最常见的初始化方式是用复合字面量，直接指定字段名和值。这样可以避免顺序错误，也更可读。
比如：
var p Person = Person{Name: "Alice", Age: 30}
或者简写成：
p := Person{Name: "Alice", Age: 30}

如果你不指定所有字段，没指定的会用零值初始化，比如字符串是空串，整数是0。

2. 如果你知道字段顺序，也可以不写字段名，按顺序赋值。但不推荐，因为结构体字段一多就容易出错。
比如：
p := Person{"Bob", 25}

3. 如果想用指针，可以用&符号：
p := &Person{Name: "Charlie", Age: 40}
这样p就是Person的指针。

4. 或者用内置的new函数分配内存，返回指针，并初始化为零值：
p := new(Person)
然后你可以单独赋值字段：
p.Name = "David"
p.Age = 35

如果你有嵌套结构体或者更复杂的情况，初始化时可以递归指定。比如另一个结构体里有Person字段，就在字面量里嵌套另一个字面量。

记得，Go里结构体是值类型，所以赋值是拷贝的。如果你用指针，就能避免不必要的拷贝。
